How to install ?

Put the tags and the javascript codes before the last </body> tag in your HTML page

<script type="text/javascript" src="http://www.stalk.io/stalk.js" charset="utf-8" ></script>
<script language="javascript">
STALK.init();
</script>

That's it! :)
